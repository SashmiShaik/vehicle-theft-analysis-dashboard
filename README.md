# vehicle-theft-analysis-dashboard

## About This Project

-I created this Excel dashboard as part of a data analytics project to understand vehicle theft patterns in New Zealand.

-The dataset contains information about stolen vehicles, vehicle manufacturers, and locations. Using Excel, I cleaned the data, created relationships between tables, and built an interactive dashboard to identify theft trends and high-risk areas.

-The main goal of this project was to turn raw data into meaningful insights that could help law enforcement agencies and insurance companies make better decisions.

---

## Questions Answered

This dashboard helps answer the following questions:

- Which region has the highest vehicle theft rate?
- Which vehicle types are stolen the most?
- How do thefts change over time?
- Are luxury vehicles targeted more than standard vehicles?
- Does population density have an impact on vehicle thefts?

## Key Insights

- Auckland recorded the highest number of vehicle theft incidents.
- Trailers were among the most frequently stolen vehicle types.
- Vehicle thefts showed noticeable variations across different months.
- Standard vehicles were stolen more often than luxury vehicles.
- Regions with higher population density generally experienced more theft incidents.
